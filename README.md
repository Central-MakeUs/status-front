# 상태창 - 육각형 인간으로 레벨업

**상태창은 실천 기반 퀘스트를 통해 나를 성장시키는 자기계발 앱입니다.** 누구나 가볍게 즐길 수 있는 자기계발 퀘스트를 통해 경험치를 얻고, 나의 능력치로 출력됩니다. 이제 자기계발을 게임처럼 즐기세요.

## 주요 기능

### 퀘스트 생성

목표로 삼고 싶은 능력치를 선택하면, 관련된 테마와 퀘스트가 자동 추천됩니다. 나만의 챌린지를 설정하고 실천을 시작하세요.

### 능력치 성장 시스템

정신 능력치(의지력, 집중력 등)와 기술 능력치(건강, 기록 등)가 실천을 통해 RPG처럼 성장합니다. 각 능력치는 경험치 기반으로 레벨업됩니다.

### 나만의 상태창 제공

지금까지 수행한 퀘스트 기록과 능력치 상태를 한눈에 확인할 수 있는 ‘상태창’ 인터페이스를 제공합니다. 성장 정체, 버닝 상태 등도 직관적으로 표현됩니다.

### 자기계발 히스토리

성장 과정은 모두 기록으로 남습니다. 꾸준한 실천은 칭호와 보너스 경험치로 보상됩니다.
